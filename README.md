# htmleditor, a terminal CLI to edit your html file
made by 凌宇航 24210240454

# running
To run the CLI, simply do
```sh
$ python main.py 
```
or
```sh
$ python main.py demo.html
```
to edit an existing html file
 
To run automated tests，please install `pytest>=7`
```sh
$ pip install pytest
$ pytest
```
